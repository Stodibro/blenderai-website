# BlenderAI Website - Production Build

This is the production build of the Blender Addon Website.

## Deployment

This folder contains the built files ready for deployment to:
- Vercel
- Netlify
- Cloudflare Pages
- Any static hosting service

## Files

- `index.html` - Main HTML file
- `assets/` - CSS and JavaScript bundles

## Updating

When you rebuild the site:
1. Run `npm run build` in the main project
2. Copy new files to this repo
3. Commit and push
